Proyek VRMPosing (React + Three.js + @pixiv/three-vrm) sudah berjalan, dengan App.jsx mengatur dropdown switch VRM model (male_00 / female_00) dan VRMViewer.jsx menampilkan model dengan loading indicator. Sebelumnya switch model menimbulkan error removeChild dan masalah load, sekarang sudah diperbaiki, model muncul normal saat diganti. Backup ada di branch Git backup_before_model_switch termasuk file VRM terbaru (vrm0_female_00.vrm, vrm0_male_00.vrm). Index.html menggunakan CSP script-src 'self' 'unsafe-eval'; supaya loader bisa berjalan.
